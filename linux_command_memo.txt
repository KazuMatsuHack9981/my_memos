linuxコマンドについて


ー基本
xdg-open : ファイルをデフォルトアプリケーションで開く
cat file : "file" の中身を表示する
grep str : "str"の文字列を含むものを探す。
	ls | grep str : "str"の文字列を含むファイルを、現在のディレクトリの子の中から探して表示
sudo apt-get instal apps : "apps"のアプリをインストールする。
sudo apt-get purge apps : "apps"のアプリをアンインストールする。
groups : 自分の所属しているグループの一覧を表示(全ユーザーは必ず何かしらのグループに属している)。
	groups name : "name"というユーザー名の所属しているグループを表示
adduser name : "name"という名のユーザーを登録
passwd name : "name"というユーザーのパスワードを変更
usermod -aG group user : "user"を"group"に追加する
id user : "user"所属グループなどの情報が確認できる
chgrp -R group directory : "directory"を"group"の共有ファイルにする
chmod -R 770 directory : ↑の次にこれをやる。




ー応用
nmap localhost : ローカルコンピュータのポートスキャンを行う
wing-personal7.1 : WING-IDEを起動する。
