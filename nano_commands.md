## cut (one line)
```
Ctl-k
```



***



## copy (one line)
```
Alt-^
```



***



## paste
```
Ctl-u
``` 



***



***



## selection start/stop
start selection by pressing
```
Ctl-^
```
then, move the cursor by arrow keys  
and press `Ctl-v` again to end selection.  
or you can  
```
Alt-^
```
to copy and end selection.



***
